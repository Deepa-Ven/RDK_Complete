ifxdaq

ifxdaq - Simplifies the entry into the acquisition of high-quality sensor data sets!

What is ifxdaq / what is provided?

  • 🐍 Access sensors in a pythonic way.

  • 🎬 Data recording tool.

  • 🤖 Automatic-generation of camera based labels.

  • 🔧 Tools to handle acquired data & generated labels.

What is ifxdaq not / what is not provided?

  • ❌ Algorithm library.

  • ❌ 100% accurate ground truth generation.

  • ❌ Manual labeling tool.

  • ❌ Framework to test or benchmark algorithms against ground truth.

  • ❌ Embedded software stack.

  • ❌ Data management tool.

ifxdaq & DaqKit

ifxdaq - Software package

DaqKit - Hardware setup

  • Python library 🐍

    >>> from ifxdaq.sensor.radar_ifx import RadarIfxBGT60TR13C

>>> print("Available devices: ", RadarIfxBGT60TR13C.discover())

>>> config_file = RadarIfxBGT60TR13C.create_default_config_file()
>>> with RadarIfxBGT60TR13C(config_file) as device:
...    for frame in device:
...        print(frame)

  • Recorder tool 🎬

  • Labeling algorithms 🤖
