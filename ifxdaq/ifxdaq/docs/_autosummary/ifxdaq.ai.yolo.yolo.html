class ifxdaq.ai.yolo.yolo.YOLO8(model='yolov8', task='det', model_size='n', classes=None, name=None, threshold=0.4, nms=0.5)

Bases: object

YOLOv8 based object detection and Tracking.

Parameters:

  • model (str) – Model to use.

  • task (Literal['det', 'seg', 'pose']) – model typem, i.e. detector, segmentation, keypoints

  • model_size (Literal['n', 's', 'm', 'l', 'x']) –

    size of model to use from ultralytics zoo, trades precision vs inference time

    • yolov8n.pt: Slim & fast.

    • yolov8s.pt: Bigger & more accurate.

    • yolov8m.pt: Bigger & more accurate.

    • yolov8l.pt: Bigger & more accurate.

    • yolov8x.pt: Largest & most accurate.

    • also allowable with suffix ‘seg’ and ‘pose’ before ‘.pt’ extension

  • threshold (float) – Detection threshold of the model.

  • nms (float) – IoU threshold for the non-maximum suppression (nms).

Raises:

RuntimeError – N/A with YOLO8: If model initialization failed with the provided weights.
